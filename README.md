# 🏥 Tele-Sağlık Platformu

Tele-Sağlık Platformu, hasta ve doktor arasındaki iletişimi dijital ortamda kolaylaştırmayı amaçlayan bir sağlık teknolojisi projesidir. Platform; uzaktan hasta takibi, online randevu sistemi ve görüntülü doktor görüşmesi gibi özellikler sunarak sağlık hizmetlerine erişimi daha hızlı ve verimli hale getirmeyi hedefler.

## 🛠️ Kullanılan Teknolojiler

- **Backend:** Java & Spring Boot  
- **Frontend:** Angular  
- **Veritabanı:** MySQL
